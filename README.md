# Spark Homes Repair Cost Estimator

## Overview

This project is a mobile-first repair cost estimator web application built for the Spark Homes Developer Contest.

The app is designed for acquisition agents who are walking through properties in the field and need to quickly estimate renovation costs before making an offer. It allows users to create property projects, add rooms, select repair items, enter quantities, override prices, attach photos, track walkthrough progress, and export the final estimate.
